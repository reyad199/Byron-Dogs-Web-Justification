Introduction

The Byron Bay Dogs website prototype has been design to be optimised and responsive specifically for desktop screens of 13 inches and Mobile (iPhone X). Due to limitations with time constraints and code knowledge, I was not able to develop fully responsive prototype across the various screen sizes.

Desktop Version 13"

Mobile Version

The mobie version of the Byron Bay Dogs website has been optimised for iPhone X but is still able to be responsive on most phone screen sizes.
